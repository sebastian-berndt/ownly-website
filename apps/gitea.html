About Gitea

Gitea is a community-driven, lightweight code hosting solution written in Go. It provides a GitHub-like experience for self-hosted environments, making it easy for teams and individuals to manage their source code, track issues, and collaborate on projects.

Despite its small footprint, Gitea offers a comprehensive set of features including repository management, code review, CI/CD, package registries, and project management tools—all without the resource overhead of larger alternatives.

  • Git repository management
  • Pull requests & code review
  • GitHub Actions compatible CI/CD
  • Built-in package registry
  • Project boards & milestones
  • GitHub/GitLab migration tools
