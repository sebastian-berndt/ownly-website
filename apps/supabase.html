About Supabase

Supabase is the leading open-source alternative to Firebase, providing developers with a complete backend platform built on PostgreSQL. Unlike proprietary solutions, Supabase gives you full access to your database and data, with no vendor lock-in.

It combines the reliability of PostgreSQL with modern developer tools including auto-generated APIs, real-time subscriptions, authentication, and serverless functions—all accessible through an intuitive dashboard.

  • PostgreSQL with row-level security
  • Built-in user authentication
  • Auto-generated REST & GraphQL APIs
  • Real-time database subscriptions
  • S3-compatible file storage
  • Dashboard with SQL editor
