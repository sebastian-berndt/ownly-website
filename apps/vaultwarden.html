About Vaultwarden

Vaultwarden is a lightweight, self-hosted implementation of the Bitwarden API, written in Rust for maximum performance with minimal resource usage. It's the ideal solution for individuals, families, and small teams who want complete control over their password management infrastructure.

Unlike the official Bitwarden server, Vaultwarden runs efficiently on low-powered hardware and includes premium features like TOTP generation, file attachments, and organization support—all completely free.

  • Zero-knowledge encryption
  • All Bitwarden clients supported
  • TOTP authenticator built-in
  • File attachments & Send
  • Emergency access
  • Minimal resource usage (Rust)
